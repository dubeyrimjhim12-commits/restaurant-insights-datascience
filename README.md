# Restaurant Insights & Data Science Project

An end-to-end Data Science project focused on analyzing a restaurant dataset to uncover business insights, customer preferences, and pricing trends. This project was developed as part of my core internship tasks.

## 📌 Project Overview
The main objective of this project is to perform comprehensive data analysis and extract actionable insights from restaurant data. It covers everything from initial data cleaning and exploration to deep visualization and behavioral analysis.

## 🛠️ Tech Stack & Libraries Used
- **Language:** Python (Jupyter Notebook / Google Colab)
- **Data Manipulation:** Pandas, NumPy
- **Data Visualization:** Matplotlib, Seaborn
- **Machine Learning (If applicable):** Scikit-Learn

## 📊 Key Analysis Performed
1. **Data Cleaning & Preprocessing:** Handled missing values, treated outliers, and removed duplicate records to ensure data quality.
2. **Exploratory Data Analysis (EDA):** Created interactive and static visualizations to analyze rating distributions, online delivery impacts, and cost dynamics.
3. **Feature Engineering:** Extracted relevant features and encoded categorical variables for deeper analytical modeling.
4. **Key Insights:** Identified top-performing restaurant types, popular cuisines, and major factors influencing high customer ratings.

## 🚀 How to Run this Project
1. Clone or download this repository to your local machine.
2. Open the `.ipynb` file in Google Colab or Jupyter Notebook.
3. Upload the restaurant dataset (`.csv` file) to your environment.
4. Run the code cells sequentially to view the analysis and charts.
